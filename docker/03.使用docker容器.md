# 03.使用docker容器

```
新建容器，交互式
docker create -it centos:8.1.1911
新建容器，后台运行
docker create -d centos:8.1.1911
启动容器，注意-d模式下启动依然是后台运行模式
docker start $CONTAINER_ID
新建并启动容器，交互式，-d后台运行
docker run -it centos:8.1.1911 /bin/bash
终止容器
docker stop $CONTAINER_ID
进入容器，容器必须启动状态，退出时终止容器
docker attach $CONTAINER_ID
进入容器，不影响容器其他应用的前提下，打开新的bash终端
docker exec -it $CONTAINER_ID /bin/bash
删除容器
docker rm $CONTAINER_ID
导出容器
docker export -o $tar-file $CONTAINER_ID
导入容器
docker import $tar-file - $REPOSITORY:$TAG

查看所有容器
docker ps -a
查看容器输出日志
docker logs $CONTAINER_ID
```
